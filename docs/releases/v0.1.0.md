SLMCortex v0.1.0 is the first public product release.

## What Shipped

- the canonical `slmcortex` product identity
- a product-first README and documentation set
- package, compose, validate, infer, serve, and bounded-agent flows
- a no-model end-to-end demo based on checked-in fixtures

## Product Workflow

```text
Slm Factory
  -> Slm Package
  -> Slm Composer
  -> Runtime Bundle
  -> Runtime Core
  -> Agent Runtime
```

## What To Try First

```bash
python3 -m venv .venv
. .venv/bin/activate
pip install -e '.[test]'
python -m slmcortex --help
python scripts/run_slmcortex_demo.py
```

## Release Boundaries

- the demo path stays dry-run first
- `compose-slms` supports the routed strategy in v0.1
- the compatibility server is intentionally minimal and non-streaming
- Agent Runtime is bounded and local, not a general IDE agent
