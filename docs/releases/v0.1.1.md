SLMCortex v0.1.1 is a patch release that makes the packaged product path
clearer without changing the core runtime model.

## Highlights

- adds the `slmcortex-composer` launcher for direct Composer app entry
- aligns default package metadata and release versioning to `0.1.1`
- makes the packaged install story easier to understand and invoke

## What Did Not Change

- runtime capabilities and adapter formats
- the dry-run-first demo story
- the minimal non-streaming compatibility server

## Why It Matters

This release improves how a new user reaches the Composer-first flow from an
installed package. It is a packaging and entrypoint refinement, not a new model
or routing feature release.
